#### CRIT-1 ÔÇö Duplicate-suppress dropped output event on crash window

- **Before**: When `ClaimExecution` / `UpsertPositionFromExecution` returned `claimed=false` on redelivery, worker returned `(nil, nil)`. #### CRIT-2 ÔÇö Unbounded retry; DLQ adapter methods existed but never invoked

- **Before**: `RunWorker` called `ReleaseEventClaim` on every handler error and continued.
